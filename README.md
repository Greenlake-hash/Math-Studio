# Math-Studio
Interactive Mathematics Visualization Dashboard featuring 2D graphing, 3D surface plots, probability simulations, and linear algebra visualizations using JavaScript, Three.js, Chart.js, Math.js, and KaTeX.

## Features

- 2D Geometry and Algebra Workbench
- 3D Surface Plotter
- Probability Distribution Simulator
- Linear Algebra Visualizer

## Technologies

- HTML5
- CSS3
- JavaScript
- Three.js
- Chart.js
- Math.js
- KaTeX

## Run Locally

npm install
npm start

Open:
http://localhost:8000
